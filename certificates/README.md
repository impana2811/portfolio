# Certificate Images Setup

Please add your certificate images to this folder with the following exact names:

1. **nptel-data-science.jpg** - NPTEL Data Science for Engineers certificate
2. **infosys-python.jpg** - Infosys Python Fundamentals certificate
3. **infosys-dsa.jpg** - Infosys Data Structures and Algorithms certificate
4. **cisco-os.jpg** - Cisco Operating Systems Basics certificate
5. **tata-data-viz.jpg** - Tata Data Visualisation certificate

## How to Add Certificates:

1. Save your certificate PDF files
2. Take a screenshot or export them as JPG/PNG images
3. Crop them to show only the certificate (no extra white space)
4. Rename them according to the names above
5. Place them in this `certificates` folder

## Image Recommendations:

- **Format**: JPG or PNG
- **Size**: Recommended width 1200-1600px for best quality
- **Aspect Ratio**: Landscape (certificate orientation)
- **File Size**: Keep under 500KB each for faster loading

Once you add the images, refresh your portfolio website to see them displayed with clickable functionality!
